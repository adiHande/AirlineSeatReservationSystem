# Airline Seat Reservation System (C)

A terminal-based airline seat reservation system in C supporting different seat layouts across flights.

## Features
- Multiple flight configurations using a master file (`Master.txt`)
- Supports different seat layouts:
  - L23 (2–3 seating)
  - L232 (2–3–2 seating)
  - L343 (3–4–3 seating)
- Dynamic seat map generation and display
- File-based seat booking persistence per flight and date
- Input validation for seat selection, flight number, and travel date
- Prevents double booking of seats
- Terminal-based interface for seat selection

## Stack & Concepts
C, File Handling, Structs & Enums, 2D Arrays, Input Validation
